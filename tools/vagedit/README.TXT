VAG Editor 1.0  ( 7 October 1998 )

This is a new tool for converting WAV, RAW and VAG files
and accurately adjusting sample data to change loop points.

Basic features:

1) Edit loop points.
2) Visual display of samples and 28 sample boundaries.
3) Conversion between RAW, WAV and VAG
4) Preview of sound on SoundBlaster card.
5) Cut and Paste samples over clipboard.
